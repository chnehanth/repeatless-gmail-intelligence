Initials-on-tint avatar — no stock photos in Repeatless. Tint color is derived deterministically from the name, so the same sender is always the same color.

```jsx
<Avatar name="Ana Ruiz" />
<Avatar name="Stripe" size="sm" square />
<Avatar name="Dev Patel" src="/p.jpg" size="lg" />
```

Sizes `xs`/`sm`/`md`/`lg` or a pixel number. `square` for rounded-square (used for brand/org senders).
