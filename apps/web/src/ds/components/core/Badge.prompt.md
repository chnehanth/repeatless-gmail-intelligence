Status / label pill. For inbox category labels use `CategoryBadge` instead; for message counts use the `count` variant (monospace).

```jsx
<Badge variant="success" dot>Synced</Badge>
<Badge variant="ai" >AI draft</Badge>
<Badge count>3</Badge>
```

Variants: neutral, blue, success, warning, danger, ai. Sizes sm/md. `dot` adds a leading status dot; `count` renders a monospace count pill.
