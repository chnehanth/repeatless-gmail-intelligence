Icon-only button for toolbars, row actions, and modal close; always pass a `label` for accessibility.

```jsx
<IconButton icon="refresh" label="Refresh" />
<IconButton icon="x" label="Close" variant="solid" size="sm" />
```

Variants: `plain` (transparent, hover tint) and `solid` (white + border). Sizes `sm`/`md`/`lg`.
